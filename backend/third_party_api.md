# Drittanbieter API

## Authentifizierung
Alle Anfragen müssen den Header `x-api-key` mit einem gültigen Schlüssel enthalten.

## Endpunkte
- `GET /api/external/health` – Prüft den API-Status.
- `GET /api/external/users/:id` – Liefert Basisinformationen zu einem Benutzer.
